# Incident Response Playbooks


## Duplicate provider callback

1. Locate raw callback.
2. Check provider event ID.
3. Check dedupe state.
4. Confirm no duplicate ledger posting.
5. Run reconciliation if needed.
6. Generate evidence.

## Payout mismatch

1. Compare internal payout state.
2. Compare provider truth.
3. Check payable and frozen funds.
4. Classify mismatch.
5. Repair through approved flow if needed.
6. Attach evidence.
