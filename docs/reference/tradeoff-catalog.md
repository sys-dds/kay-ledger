# Trade-off Catalog

| Decision | Trade-off |
|---|---|
| Modular monolith | simpler correctness, less deployment flexibility |
| PostgreSQL | strong source truth, scaling needs care |
| Outbox/inbox | more tables, safer async |
| Temporal | extra runtime, durable workflows |
| OpenSearch | powerful search, derived not truth |
| Multi-region simulation | realistic concepts, not full production global finance |
